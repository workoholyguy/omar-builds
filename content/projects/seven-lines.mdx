## Overview

A production corporate website for Seven Lines Inc., a logistics and delivery company, built with modern web technologies and optimized for lead generation.

## Problem

The client needed a professional web presence that clearly communicated their services, built trust with potential customers, and facilitated easy contact.

## Solution

Developed a modern, responsive website with clear service hierarchy, optimized performance, and integrated contact forms for lead capture.

## My Contributions

- Developed complete website using Next.js App Router architecture
- Implemented mobile-first responsive layouts
- Integrated production email forms with environment-based configuration
- Applied static optimization for fast page loads
- Improved navigation clarity and visual hierarchy

## Technical Details

Next.js 14 with App Router provides server-side rendering and automatic code splitting. TypeScript ensures type safety. Tailwind CSS enables rapid, consistent styling with minimal CSS overhead.

## Challenges & Tradeoffs

**Challenge:** Balancing visual richness with page load performance on mobile networks.

**Solution:** Used Next.js image optimization, lazy loading, and static generation to achieve fast loads while maintaining visual quality.
